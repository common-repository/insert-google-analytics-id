This is a simple plugin to insert your Google Analytics Tracking Id. This plugin hooks the analytic code directly to the header section.

== Description ==

Inserting the google Tracking Id to your website is made simple by **Insert Google Analytics Id**. It is a simple but powerful tool. 

= Main Features =

* Simple and user friendly interface.
* No programming knowledge required to implement.
* No header or function files modification required.
* Hooks your Tracking ID to the head section directly.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/insert-google-analytics-id` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->Plugin Name screen to configure the plugin.
